## Laravel projekt beállítása

- `git clone git@github.com:anonymus1928/Szerveroldali.git`
- `composer install`
- `npm install`
- `npm run build`
- `cp .env.example .env`
- `touch database/database.sqlite`
- `php artisan key:generate`
- `php artisan migrate:fresh --seed`
- `php artisan serve`
